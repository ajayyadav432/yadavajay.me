# Ajay Yadav - Personal Portfolio

A modern, responsive portfolio website showcasing my skills and journey as a computer science student with a focus on Web3 and blockchain development.

## 🌟 Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, professional interface with smooth animations
- **Interactive Elements**: Animated backgrounds, hover effects, and smooth scrolling
- **Web3 Focused**: Showcases blockchain and decentralized application projects
- **Performance Optimized**: Fast loading times and efficient code structure

## 🚀 Live Demo

Visit my portfolio: [yadavajay.me](https://yadavajay.me)

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Icons**: Font Awesome 6.4.0
- **Fonts**: Google Fonts (Fira Code, Space Grotesk)
- **Animations**: CSS3 keyframes and transitions
- **Design**: Responsive grid layouts with CSS Grid and Flexbox
